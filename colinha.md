# Lista de consulta rápida para comandos utilizados em sala

Esses comandos devem ser digitados NO TERMINAL

Substitua o que estiver entre <> pelo nome do item desejado

### Criação e mudança de pastas/diretórios

- Criar uma pasta: `mkdir <nome da pasta>`
- Acessar uma pasta: `cd <nome da pasta>`

### Comandos para executarmos tarefa no próprio Django

- Criar um novo projeto: `django-admin startproject <nome do projeto>`
- Iniciar o servidor: `python manage.py runserver`
- Criar um aplicativo: `python manage.py startapp <nome do app>`
- Criação do superusuário no painel de administrador: `python manage.py createsuperuser`

- Encerrar o servidor: `CTRL + C`

### Comandos referentes as migrações e operações relacionadas ao banco de dados

- Criar as migrações: `python manage.py makemigrations`
- Executar as migrações: `python manage.py migrate`
