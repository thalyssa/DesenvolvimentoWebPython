# Aula 01 - Introdução ao conceito de aplicação web e preparação do ambiente

## Para fazer o projeto proposto neste curso precisaremos instalar...

- VSCode: https://code.visualstudio.com/Download
- Python: https://www.python.org/downloads/
- Django: https://www.djangoproject.com/download/

Git não será abordado no escopo deste curso, mas a instalação e utilização é recomendada caso você queira uma maneira de versionar o seu código e enviá-lo para um repositório remoto depois.

Caso você tenha alguma dúvida, seguem alguns tutoriais em vídeo para auxiliar na instalação.
